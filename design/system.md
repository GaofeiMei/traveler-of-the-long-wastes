# 人物属性

## 职业
每个角色具有一个职业，可转职刷属性，要有限制
1. 赏金猎人（全能型）
2. 战士（热武器大师）
3. 机械师（战车型）
4. 格斗家（空手搏斗型）
6. 护士（治疗型）

## 战斗属性
1. 力量：影响冷兵器攻击力
2. 技巧：影响热武器攻击力
3. 速度：影响行动顺序，每回合开始所有角色选择动作，然后按顺序进行
4. 体力：影响生命值
5. 驾驶：影响驾驶战车的性能（不同战车有不同驾驶难度，需要一个公式协调驾驶值和驾驶难度来决定发挥多少战车性能）
6. 运气：影响会心一击，修正命中率和闪避率，修正掉落

## 防御力
防御力以抗性的形式展现，使用数值而非百分比：
1. 实体抗性
  - 爆炸
  - 穿刺
  - 钝击
  - 挥砍
3. 能量抗性
  - 火焰
  - 冰冻
  - 闪电
  - 毒素
  - 音波
  - 激光
8. 特效抗性
  - 燃烧（持续伤害，伤害较高，持续时间较短，受冰冻攻击解除）
  - 冻结（无法行动，实体抗性下降，受火焰攻击解除）
  - 麻痹（概率性无法行动，持续时间较长）
  - 睡眠（无法行动，受到伤害增加，仅对生物系有效，受击解除）
  - 中毒（持续伤害，伤害较低，持续时间较长）
  - 眩晕（部分能力下降，并非控制）
  - 目盲（命中下降）
  - 减速（速度下降）

## 命中
1. 基础命中率+技能命中率修正
2. 基础闪避率
