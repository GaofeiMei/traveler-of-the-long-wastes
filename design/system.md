# 游戏类型

2D jrpg，回合制战斗，暗雷遇敌，部分boss、精英为明雷，触发战斗时切换到战斗场景，敌人在屏幕左边，我方在右边。

# 人物属性

无战车战，只有一辆旅行车用于移动、修整。

## 职业

每个角色具有一个主职业，可随剧情推进开启一个副职，副职只提供技能。

1. 探险家（全能型）
2. 佣兵（热武器大师）
3. 格斗家（空手搏斗型）
4. 护士（治疗型）

## 战斗属性

1. 力量：影响冷兵器及徒手攻击力
2. 技巧：影响热武器攻击力
3. 速度：影响行动顺序，每回合开始所有角色选择动作，然后按顺序进行
4. 体力：影响生命值
5. 运气：影响会心一击，修正命中率和闪避率，修正掉落

## 防御力

防御力以抗性的形式展现，使用数值而非百分比：

1. 实体抗性
  - 爆炸
  - 穿刺
  - 钝击
  - 挥砍
2. 能量抗性
  - 火焰
  - 冰冻
  - 闪电
  - 毒素
  - 音波
  - 激光
3. 特效抗性
  - 燃烧（持续伤害，伤害较高，持续时间较短，受冰冻攻击解除）
  - 冻结（无法行动，实体抗性下降，受火焰攻击解除）
  - 麻痹（概率性无法行动，持续时间较长）
  - 睡眠（无法行动，受到伤害增加，仅对生物系有效，受击解除）
  - 中毒（持续伤害，伤害较低，持续时间较长）
  - 眩晕（部分能力下降，并非控制）
  - 目盲（命中下降）
  - 减速（速度下降）
  - 混乱（单位随机行动，不受控制，可能会攻击友军）
  - 恐慌（无法做出攻击类动作，行动顺序随机提前或延后）

## 命中

1. 基础命中率+技能命中率修正
2. 基础闪避率
